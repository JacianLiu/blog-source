服务器与Web页面交互方式

传统的 Web 服务都是客户端发出请求，服务端给出响应。

HTTP

  • HTTP短连接：在HTTP1.0中，客户端发送请求，服务器接收请求， 双⽅建⽴连接，服务器响应资源，请求结束。
  • HTTP⻓连接：在HTTP 1.1中，客户端发出请求，服务端接收请 求，双⽅建⽴连接，在服务端没有返回之前保持连接，当客户端再 发送请求时，它会使⽤同⼀个连接。这⼀直继续到客户端或服务器 端认为会话已经结束，其中⼀⽅中断连接。
